# Alt-SchoolBackend-Engineering-Assignment-2__User-Registration-Form

## Overview
This project was created to fulfill the requirements of the second assignment of my ALT School Backend Engineering Diploma program.

The project focuses on building a user registration form with JavaScript validation to ensure that user inputs meet the required conditions before form submission.

---

## Date
10th May, 2026

---

## Instructions
Build a user registration form that validates user inputs using JavaScript before submission.

The form should include the following fields:

- Full Name
- Email Address
- Password
- Confirm Password
- Age

---

## Validation Requirements

| Field | Rule |
|------|------|
| Full Name | Must not be empty and must contain at least 2 words |
| Email Address | Must follow a valid email format (e.g. example@domain.com) |
| Password | Must contain at least 8 characters, one uppercase letter, one number, and one special character |
| Confirm Password | Must match the Password field |
| Age | Must be 18 years or older |

---

## Functional Requirements

The application should include the following features:

- Alert error messages when an input is invalid
- Prevent form submission if there are validation errors
- Show a success message if all inputs are valid

---

## Additional Features

- Password strength indicator
- Show/Hide password functionality
- Real-time input validation
- Styled success modal
- Responsive design layout
